# PlanSaludGratis
App gratuita BMI + plan salud WhatsApp lista para monetizar. Está diseñada específicamente para desplegar como página estática usando HTML5, CSS3, y JS Vanilla.

## 🚀 Deploy GitHub Pages
1. Fork este repo o crea uno nuevo llamado `plan-salud-gratis`
2. Sube estos 5 archivos a la rama `gh-pages`
3. Ve a Settings → Pages → Source: selecciona la rama `gh-pages`
4. Live: https://[tuusuario].github.io/plan-salud-gratis/
5. Adsense: Reemplaza las etiquetas `[ADSENSE ...]` dentro de los divs `.ad-container` por tus scripts de anuncios.

## Estructura
- `index.html`: La vista de la calculadora.
- `css/style.css`: Archivo de estilos responsivos.
- `js/script.js`: Archivo con la lógica de cálculo (BMR, BMI, TDEE, etc.) y la integración de WhatsApp.
- `404.html`: Para no perder tráfico.
